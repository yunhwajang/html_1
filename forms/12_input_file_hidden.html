파일 타입은 일반 input과 속성이 다름

파일이 첨부된 form은 enctype의 값이 multipart/form-data를 지정해야 함

파일전송은 반드시 post방식으로 전송해야 한다(전송데이터 크기의 한계)





file type은 보안정책상 수정관련 기능을 제공하지 않는다

value 속성값에 기존 정보를 로딩할 수 없다

hidden 타입은 화면에 출력되지 않은 상태에서 데이터 전송이 필요할 때 사용 - 많은 부분에서 사용이 되므로 꼭 알아두실 것!
